# CardsMaker v1.3

## Major features
- SVG editable model (for now only one model is available)
- Text editor:
  - Text __content__
  - Text __size__
  - Text alignment
- Image editor:
  - Input file
- Download the custom model (with custom texts and images) as a raster file (.png)
- Multiple cards models
- Save cards for future uses
